# KZ VoiceShield

AI anti-scam call review MVP for Kazakh/Russian conversations.

## Features

- Real-time browser speech-to-text demo
- Scam risk scoring from transcript
- Detection of bank impersonation, SMS/PIN requests, money transfer pressure, urgency, isolation, unofficial channels and remote access prompts
- Case summary with recommended action
- Downloadable text report

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
